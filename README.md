# api-automation-exercise

## Built with
* Javascript
* supertest
* mocha
* chai

How to run:
 * Clone Repository
### `https://github.com/dwiana161/api-automation-exercise.git`

## Install the project
* `npm i -g mocha`
* `npm init`
* `npm i supertest`
* `npm i chai`
* `npm i jsonschema`

## Run the project
* `npx mocha test/suite/user.suite.js`
